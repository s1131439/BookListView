# BookListView

## 簡介

簡易的圖書借閱系統，支援借閱與歸還。

系統提供：

- 書籍清單顯示
- 多種 ListView 檢視模式
- 借閱功能
- 防止重複借閱
- 歸還功能

書籍內容以世界文學名著為主，例如：

- 《木馬屠城記》
- 《格列佛遊記》
- 《老人與海》
- 《白鯨記》
- 《金銀島》

---

# 使用說明

## 書籍顯示

程式啟動後：

- 左側顯示所有書籍
- 包含：
  - 書名
  - 作者
  - 類別

---

## 檢視模式切換

切換不同顯示方式：

- 大圖示
- 詳細資料
- 小圖示
- 清單
- 大圖示加詳細資料

---

## 借閱功能

### 操作方式

- 在書籍清單中「雙擊」或「單擊選中並按下Enter」想借閱的書籍
- 系統會跳出確認視窗：

```text
確定要借閱嗎?
```

- 按下「是」或「鍵盤Y」後：
  - 書籍會加入借閱清單
- 按下「否」或「鍵盤N」後：
  - 書籍不會加入借閱清單
---

## 防止重複借閱

若書籍已存在於借閱清單：

- 不會重複新增
- 避免同一本書被借閱多次

---

# 新增功能：歸還功能

## 功能說明

新增「歸還書籍」功能，讓使用者可以將已借閱的書籍從借閱清單中移除。

---

## 操作方式

- 在借閱清單中
- 對想歸還的書籍「雙擊」

系統會跳出確認視窗：

```text
確定要歸還嗎?
```

- 按下「是」或「鍵盤Y」後：
  - 書籍會從借閱清單移除
- 按下「否」或「鍵盤N」後：
  - 書籍不會從借閱清單移除
---


# 程式特色

- 使用 `ListView` 顯示多欄資料
- 使用 `ComboBox` 切換檢視模式
- 使用 `MessageBox` 進行借閱確認
- 使用 `ListBox` 管理借閱清單
- 具備借閱與歸還功能
- 防止重複借閱
